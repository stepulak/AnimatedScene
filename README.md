Animated scene written in C++ using OpenGL, GLM, GLUT and DevIL libraries.

School project for PV112.

Features:

Arcball camera

Point and spot lights

"Magnifying" mirror

Primitive physics and boucing

Plenty of .OBJ files

Procedural textures (counted in shaders during rendering)

Animations

Rubik Cube

Exponential fog

etc...
